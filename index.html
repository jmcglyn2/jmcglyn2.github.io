Pokemon Base Stats: Powercreep and Design Decisions Throughout the Generations¶

Jonah McGlynn¶

You would be hardpressed to find someone who isn't aware of pokemon, whether they grew up watching the show and playing the games, seen the giant pikachu baloon on Thankgiving day, or have vaguely heard that the most recent games weren't all that great. Pokemon has been around since 1996 and has been releasing games continuously ever since, with the most recent games coming out just this past month. The games are divided into generations, of which Scarlet and Violet, the ones that came out last month, mark the beginning of the 9th. Each generation is defined by the new set of Pokemon they introduce, this can be as low as 78 new Pokemon or as high as 151. With so many Pokemon, it is inevitable that some end up better than others, however, some people make the argument that Pokemon has undergone powercreep over the years.

Powercreep is defined as the gradual unbalancing of a game due to successive releases of new content, leaving the older ones underpowered. In this case it would mean that new pokemon that are released are on average more powerful than those from earlier generations. But how do we define a more powerful pokemon? Pokemon have many stats that determine how they perform. They have stats for Health, Attack, Defense, Sp. Attack, Sp. Defense, and Speed. When playing the game how these grow can seem random, and it is intentionally designed to be so with complicating factors that we won't discuss in depth like EVs, IVs, and natures. However there is a fundamental driving force intrinsic to each Pokemon that determines how they develop. These are called base stats, they correspond to each of the stats discussed before, the higher the base stat, the higher the growth rate, and potential maximum a given stat can be. So a more powerful pokemo can generally be described as a pokemon with a higher base stat total. In order to determine whether or not powercreep is actually occuring we must observe base stat totals over the generations to see if and how they improve.

Another argument occasionally made is that certain generations tend to introduce pokemon with a tendency towards certain stats. It, for example, is often said that generation 4 is a slow and bulky generation as it is percieved as having a lot of pokemon with low speed and high defenses compared to other generations. In order to test the truth about this, we can look at the averages for each generation and using statistics make decisions about whether there is any significant differences between the generations for given stat types (offensive, defensive, and speed).

In order to get this data I found a dataset using kaggle, unfortunately I had trouble scraping it directly from python so I am putting a link of it here: https://www.kaggle.com/datasets/mariotormo/complete-pokemon-dataset-updated-090420

Although a line has been found we need to test if the supposed increase is significant. Using statsmodel api we use the least squares method to test the hypothesis that the slope is significantly different from 0 for the formula relating Generation number to base stat increase (we assume a relatively constant increase per generation on avergae).

The analysis shows an r squared value of only .024 meaning the line does not explain the variation very well, this is understandable however due to the massive variation within a generation, and it does not necessarily invalidate our model. Looking further down we see the coefficients for the regression being 400.732 as the intercepet and 8.2623 as the slope, which means (assuming that we can find the coefficient is statistically significantly different from 0), that the average base stat increase is about 8.2623 per generation.

We see the slope (next to generation as slope is the coefficient of generation in the formula) has a t value of 5, very high, which corresponds to a p value of 0. Since the p value is so low we can reject the null hypothesis. As such given the model is valid, we see about an 8.2 point increase in base stats per generation.

2. Design Decisions¶

First I decided to explore this. I plotted average attack and special attack combined to get average offensive power, average hp, defense, and sp defense to get average defense, and speed for, well, speed. I chose to use a bar graph as I felt it gave a good visualization for differences in mean. I did this for all pokemon and for no megas as, as previously shown, the distributionof stats in all pokemon is not normal. I chose no mega as choosing all FE would only have buffed gen 6 without doing much else, I felt this was a better way of showing generational philosophies for pokemon individually rather than battle buffs.

Conclusion¶

As we have shown the slope for increase in base stat totals is always significantly different from 0, and the average new non mega fully evolved pokemon is shown to have an average increase of 6 point sover the previous generation. So if we base a pokemons power solely on its BST then powercreep is indeed occuring.

Further, it seems pokemon stat design is more similar among generations than is commonly believed. Average speed stats could not be shown to have a significant difference between generations, the only defenseive difference occured between generation 1 and 6, and offensely mostly showed that generation 2 was weaker than most of its peers.
